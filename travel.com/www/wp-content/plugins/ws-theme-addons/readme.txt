== Description ==

Provides Additional Features and functionalities for themes. Provides one click demo importer functionality for themes from WEN Solutions.

== Features ==
* One Click Demo Import for Travel Log theme(https://wordpress.org/themes/travel-log/).
* Widgets
    * WS Instagram Widget

== Installation ==

= Uploading in WordPress Dashboard =
* Navigate to the 'Add New' in the plugins dashboard
* Navigate to the 'Upload' area
* Select ws-theme-addons.zip from your computer
* Click 'Install Now'
* Activate the plugin in the Plugin dashboard

= Using FTP =
* Download ws-theme-addons.zip
* Extract the ws-theme-addons directory to your computer
* Upload the ws-theme-addons directory to the /wp-content/plugins/ directory
* Activate the plugin in the Plugin dashboard
